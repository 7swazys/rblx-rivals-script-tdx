# Remzy/TDX - Keybinds Documentation

## Keybind Reference

| Key | Function | Description |
|-----|----------|-------------|
| R | Aimbot Toggle | Enables or disables the aimbot functionality. When enabled, automatically aims at the nearest enemy player within range. |
| K | Mode Switch | Toggles between Smooth mode (1) and Snappy mode (2). Smooth mode uses gradual aiming, Snappy mode provides faster target acquisition. |
| J | Rage Mode | Activates Rage mode (3) with a larger field of view (400) for aggressive targeting. Press again to return to normal mode. |
| [ (Left Bracket) | Method Toggle | Switches between MouseRel (relative mouse movement) and CFrame (camera position) aiming methods. |
| ] (Right Bracket) | Auto-Click / Triggerbot | Enables automatic firing when aiming at a valid enemy target. The script will simulate mouse clicks at random intervals. |
| G | Auto-Collect | Automatically collects nearby drops including health packs and ammunition from objects named "_drop". |
| B | Flight Mode | Toggles flight capability. Use W/A/S/D to move directionally and Space to ascend. |
| N | Noclip | Toggles noclip mode, allowing the player to pass through solid objects and walls. |
| E | TPWalk | Enables teleport walking. When active, movement is significantly faster than normal walking speed (set to 45 speed). |
| L | Infinite Jump | Enables infinite jumping. While active, the player can jump repeatedly in mid-air with increased jump height. |

## Additional Controls

| Action | Description |
|--------|-------------|
| W/A/S/D | Directional movement (works with Flight mode) |
| Space | Jump / Ascend while flying |

## Notes

- All keybind settings are automatically saved to `tdx_config.json` and persist between game sessions
- The aimbot field of view circle is visible when aimbot is enabled and visuals are turned on
- Different aiming modes have different field of view settings: Smooth (10), Snappy (150), Rage (400)
